# Auto Hotkey Scripts

## Introduction

This repository contains a collection of AutoHotkey scripts designed to automate repetitive Windows tasks, streamline workflows, and improve productivity.  
The scripts cover scenarios such as data entry automation, application navigation, and batch processing for business operations.

## Documentation & Resources

- Official AutoHotkey website: https://www.autohotkey.com  
- AutoHotkey v2 documentation: https://www.autohotkey.com/docs/v2/

## delete-myob-purchase.ahk
AutoHotkey v2 automation script that reads purchase codes from an Excel file and processes them in MYOB AccountRight.  
For each code, the script searches the purchase, opens the record, prompts for confirmation, and deletes or skips it.  
Designed for semi-automated batch cleanup with manual verification and hotkeys for pause or exit.
